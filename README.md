✨ What Makes This Special?

🚀 This is not just a basic 2048 clone — it focuses on:
	•	⚡ Smooth gesture-based gameplay
	•	🧠 Smart tile merging logic
	•	💾 Persistent game state (Continue anytime)
	•	📊 Statistics tracking (Best Score & Moves)
	•	🎨 Modern gradient UI for premium feel
	•	🔄 Seamless navigation & user experience

  🎮 Features
	•	🕹️ Swipe in 4 directions (Left, Right, Up, Down)
	•	🔢 Dynamic tile merging (2048 logic)
	•	🎯 Real-time score updates
	•	💾 Auto-save after every move
	•	🔁 Continue game from last state
	•	🆕 Start a fresh new game anytime
	•	📊 Statistics screen:
	•	Best Score
	•	Move tracking
	•	🔙 Back navigation support

🧠 Game Logic Explained
	•	Each swipe triggers:
	•	Tile movement
	•	Merge of identical tiles
	•	Score update
	•	New tile spawn (2 or 4)
	•	Game state is stored locally after every move
	•	Resume functionality restores:
	•	Board state
	•	Score
