🧠 Game Logic Explained
	•	Each swipe triggers:
	•	Tile movement
	•	Merge of identical tiles
	•	Score update
	•	New tile spawn (2 or 4)
	•	Game state is stored locally after every move
	•	Resume functionality restores:
	•	Board state
